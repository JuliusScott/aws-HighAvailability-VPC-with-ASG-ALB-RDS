# AWS High Availability VPC with ASG, ALB, and RDS

Built a custom AWS High Availability environment using Amazon VPC, Application Load Balancer (ALB), Auto Scaling Group (ASG), Amazon EC2, Amazon RDS, and AWS Systems Manager Session Manager.

This project demonstrates VPC networking, security groups, IAM roles, high availability design, load balancing, database connectivity, and infrastructure validation through the AWS Console and Linux command line.

The repository includes architecture diagrams, deployment steps, validation screenshots, and lessons learned throughout the build process.
